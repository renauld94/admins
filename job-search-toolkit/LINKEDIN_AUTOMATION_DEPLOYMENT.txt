## Troubleshooting

### Issue: "No connections being sent"

**Diagnosis:**
```bash
tail -20 outputs/logs/linkedin_daily_$(date +%Y%m%d).log | grep -E "ERROR|WARNING"
```

**Common causes:**
- Database locked (another process using it)
- Rate limit already hit today
- Missing profile configuration

**Solution:**
```bash
# Check limits
/usr/bin/python3 -c "from linkedin_network_growth import LinkedInNetworkGrowth; print(LinkedInNetworkGrowth().get_daily_limits())"

# If limits reached, wait until next day or manually reset:
sqlite3 data/linkedin_contacts.db "DELETE FROM daily_activity WHERE date = DATE('now');"
```

### Issue: "Cron job not running"

**Verification:**
```bash
# Check if cron is installed and running
sudo service cron status

# Verify cron job exists
crontab -l | grep linkedin

# Check cron logs (Linux)
sudo grep CRON /var/log/syslog | tail -20
```

**Solution:**
```bash
# Re-add to cron
crontab -l > /tmp/cron_backup.txt
echo "15 7 * * * cd /home/simon/Learning-Management-System-Academy/job-search-toolkit && /usr/bin/python3 daily_linkedin_outreach.py >> outputs/logs/cron_linkedin_daily.log 2>&1" >> /tmp/cron_backup.txt
crontab /tmp/cron_backup.txt
```

### Issue: "Email not receiving summaries"

**Check configuration:**
```bash
grep -E "PRIMARY_EMAIL|BACKUP_EMAIL" linkedin_network_growth.py

# Should show:
# PRIMARY_EMAIL = "contact@simondatalab.de"
```
