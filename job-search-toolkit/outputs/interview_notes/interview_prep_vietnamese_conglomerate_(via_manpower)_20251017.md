### Technical Concepts

1. Explain the difference between ETL and ELT. When would you use each?
2. How would you design a data pipeline to process 1 billion records daily?
3. What is data partitioning? How does it improve query performance?
4. Explain slowly changing dimensions (SCD) types 1, 2, and 3.
5. How would you handle late-arriving data in a streaming pipeline?


### Technical Sql

1. Write a query to find the second highest salary from an Employee table.
2. How would you find duplicate records in a table?
3. Write a query to calculate a 7-day moving average.
4. Explain window functions. Provide an example using ROW_NUMBER().
5. How would you join three tables efficiently?

1. How would you read and process a 10 GB CSV file in Python?

5. Explain decorators in Python. Write a simple example.
