# Assessments & Quizzes - J&J Python Academy

## Overview
Each module includes formative quizzes and a final assessment to reinforce learning and test practical skills.

---

## Module 1: System Setup & Introduction
- **Quiz:**
    - Multiple choice: Python installation steps
    - Short answer: Benefits of virtual environments
    - True/False: Jupyter is only for Python

## Module 2: Core Python Programming
- **Quiz:**
    - Code completion: Write a function to sum a list
    - Multiple choice: Data structure selection
    - Debugging: Identify errors in sample code

## Module 3: PySpark
- **Quiz:**
    - Multiple choice: RDD vs DataFrame
    - Short answer: SparkSession purpose
    - Code: Filter DataFrame rows by condition

## Module 4: Git & Bitbucket
- **Quiz:**
    - Multiple choice: Git command usage
    - Fill in the blank: Branching workflow
    - Scenario: Resolve a merge conflict

## Module 5: Databricks
- **Quiz:**
    - Multiple choice: Databricks features
    - Short answer: dbutils usage
    - Code: Run a PySpark job in a notebook

## Module 6: Advanced & Capstone
- **Final Assessment:**
    - Project: Build and present a clinical data pipeline
    - Peer review: Evaluate another student's project
    - Reflection: Lessons learned and next steps

---

## Grading Rubric
- Quizzes: 40%
- Capstone Project: 40%
- Participation/Peer Review: 20%
