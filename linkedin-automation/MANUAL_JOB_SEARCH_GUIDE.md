## üö® **RED FLAGS** (Skip These Jobs)

### ‚ùå Don't Apply If:
1. **Tech stack mismatch**: 
   - Java/C# only (you're Python-first)
   - .NET, SAP, Oracle (legacy)
   - No mention of modern data stack (Airflow, dbt, Spark)

2. **Vague job description**:
   - "Wear many hats"
   - "Fast-paced startup" (pre-seed)
   - No clear tech stack listed
   - "Rockstar", "Ninja", "Guru" (unprofessional)

3. **Location issues**:
   - On-site only in obscure location
   - No visa sponsorship mentioned (for SG/AU roles)
   - Requires relocation immediately

4. **Experience mismatch**:
   - Requires PhD
   - 15+ years experience (overqualified)
   - Entry-level (underqualified)
