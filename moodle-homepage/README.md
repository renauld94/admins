# NeuroData Science ‚Äî Homepage

This repository contains the interactive homepage for the NeuroData Science Learning Management System. The site provides a professional entry point to the platform, showcases visual analytics, and documents the primary learning domains and tools used in our curriculum.

The content is written and organized so it can be used as a knowledge source for automated assistants (for example, a LLaMA-based question-answering endpoint) and for human readers.

## What this site provides

- An overview of the learning platform and direct link to the hosted Moodle instance.
- Interactive visualizations that communicate relationships between neuroscience, data science, geospatial analysis, and research methods.
- Lightweight demo datasets and visual patterns useful for teaching and illustrating concepts.

## Key Visualizations and Components

- Knowledge network (D3.js): Shows relationships between domains such as Neuroscience, Data Science, Machine Learning, Statistics and Ethics.
- 3D brain regions (Three.js): Simplified representation of functional brain areas used for illustration.
- Geospatial map (Leaflet): Locations of learning hubs and city-level intelligence metrics.
- Mathematical visualizations: Golden ratio spirals, chaos attractors and fractal trees.
- Emotional intelligence dashboard: Visual indicators and analytics for EQ-related concepts.

## Technology stack

- D3.js, Three.js, Plotly.js, Chart.js, Leaflet, GSAP, p5.js.
- The page loads libraries via CDN for simplicity; in production these can be bundled.

## How to run locally

1. From the `moodle-homepage` directory, start a simple HTTP server:

```bash
python3 -m http.server 8000
```

2. Open `http://localhost:8000` in a modern browser.
